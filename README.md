# Project Dalek's World  

## A very simple chat application made by me as a part of my practice of coding in python and web hosting.

Even though it is not that safe or secure, the app is very fast for it's sipicity .If you are bored texting through traditional messenging apps and want to host a chat system temporarily just for fun, you can use the code.The app features:

* discord-like experience
* has 3 channels (you can always add more or less by tweaking the code)
* one of the channels is password protected.(called secret channel)
* has a hardcoded admin username (@ashrafulishti) with access to the admin page, from where we can change/create password for the secret channel)
* only text-based.
 ## important notes : 
 * after creation/ hosting , register a profile using the username @ashrafulishti, and then use this profile to change secret channel password.

To get an idea about how things look like ,you can visit https://dalek-s-world.onrender.com/register .Register and login to chat.
